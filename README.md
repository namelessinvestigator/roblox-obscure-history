# Roblox Obscure History Archive

This repository documents obscure parts of Roblox exploit history,
including R3tc, Plasmablight, Team c00lkidd, the Brample Notes,
and other community artifacts.

## Goals

- Preserve screenshots and files
- Separate evidence from rumors
- Encourage additional witnesses to contribute
- Archive material before it disappears

## Important

This repository does not assume every claim is true.

Each entry includes evidence and a confidence rating.
